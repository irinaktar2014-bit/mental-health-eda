# Mental Health EDA Project

## Objective
Exploratory Data Analysis on mental health survey data to understand patterns, trends, and factors affecting mental health in the workplace.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

  ## Key Insights
  - Mental health treatment varies by gender
  - Family history plays an important role
  - Younger professionals are more open about mental health

    ## Conclusion
    Mental health awareness is increasing, but stigma still exists. Organizations should create supportive environments.
